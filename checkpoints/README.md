# Checkpoints

Place SAM2.1 checkpoint files in this folder.

The `*.pt` files are intentionally ignored by git because they are large model artifacts.

Use:

```powershell
.\scripts\download_checkpoints.ps1
```

Expected filenames:

```text
sam2.1_hiera_tiny.pt
sam2.1_hiera_small.pt
sam2.1_hiera_base_plus.pt
sam2.1_hiera_large.pt
```
